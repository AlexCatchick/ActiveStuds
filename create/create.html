Table to CSV

Welcome to the Dynamic Table application! Here you can create a custom table by adding rows and columns, enter data into the cells, and export the table data as a CSV file.

How to use:

  1. Click on the "Add Column" button to add a new column to the table. Enter the column name and select the data type from the dropdown.
  2. Once columns are added, you can click on the "Add Row" button to add a new row to the table.
  3. Click on any cell in the table to edit its value. You'll be prompted to enter a new value.
  4. Once you've entered all the data into the table, click on the "Export CSV" button to download the table data as a CSV file.
